

# Commands
~~~bash
# lookup command helps
bin/monit-agent help

# test connection to master host
bin/monit-agent test

# run in foreground
bin/monit-agent 

# run in background
bin/monit-agent start

# stop agent
bin/monit-agent stop

# restart agent
bin/monit-agent restart 

# run a task
bin/monit-agent run [TASK ID]

# run a monitor item data source
bin/monit-agent run [ITEM ID]
~~~
